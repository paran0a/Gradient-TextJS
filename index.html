GRADIENT-TEXTJS

Gradient your text with ease

Github project page
  • Does not hinder your site performance
  • Extremely lightweight (less than 600bytes)
  • No dependencies
  • Supports IE8+, FireFox, Chrome , Opera
How?
Each letter will get wrapped in a span with its specific color creating a illusion of a gradient.
Usage
Declare your element and pick desired colors. Function accepts HEX color codes.
            var el = document.getElementById('gradMe');
            gradText(el, '#00BB99', '#FF5599');
